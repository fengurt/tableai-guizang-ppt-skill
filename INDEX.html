02 · 30 秒开始

复制一个模板，填入内容，直接打开。

没有构建，没有服务器，没有依赖。Python 任意一行 http.server 就能跑。 三个模板都自带同一份 Q3 demo，可以直接对比视觉效果。

触发关键词

Claude Code 看到 "杂志风 PPT" / "瑞士风 PPT" / "Table AI" / "horizontal swipe deck" 都会自动调用本 Skill。

不要改

Style C 不应发明任何 S23 / S24 编号 — 所有版式变更必须先在 Style B 上游合入。

不要用户自定义 hex 值 — 主题色固定保护美学稳定性。

04 · 校验器

内置两个 lint 脚本，CI 友好。

两个 validator 共享 S01-S22 版式白名单。Style C 的 validator 还额外检查 Manrope 字重、box-shadow 禁用、hex 色板白名单。

Style B 校验规则
  • · data-layout 必须在 S01-S22 / SWISS-COVER-ASCII / SWISS-CLOSING-ASCII 内
  • · 顶部大标题不得 text-align:center
  • · SVG 不得包含可见 <text>
  • · S22 必须使用 data-image-slot="s22-hero-21x9"
  • · object-position:top center 在 S22 照片上禁用 (会裁人脸)
Style C 额外规则
  • · 主标题不得用 font-weight:200/300 (应保持 600/700)
  • · 卡片不得使用 box-shadow
  • · hex 不得超出 ink / gold / paper / grey ramp 白名单
